# Focus Timer

Simple macOS menu bar focus timer for personal use.

## Features

- 30-minute default interval
- Presets: 30 minutes and 1 hour
- Custom minutes/seconds interval input
- Start, pause/resume, and stop controls
- macOS notification reminders with sound

## Run

From the project root:

```bash
./beginapp
```

Build + open app:

```bash
./beginapp --open
```

The built app is at:

`dist/FocusTimer.app`
